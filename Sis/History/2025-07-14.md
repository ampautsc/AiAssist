# July 14, 2025 - Sis History

## Session: Feat Path Tradeoff UI Update

### What Happened
Creator made a key insight about the universal CHA math for Lore Bard builds:
- Every species with Tasha's flexible +2 starts at CHA 17 (15 point buy + 2 racial)
- One CHA half-feat → CHA 18 (DC 16), leaving one feat slot for defense
- Half-feat + ASI → CHA 20 (DC 17), but BOTH feat slots consumed = zero concentration protection
- The tradeoff: higher DCs vs ability to hold concentration spells

### Actions Taken
1. **Redesigned BuildsPage.jsx** to make this the central organizing principle:
   - Added "The Core Tradeoff" explainer panel at the top of the builds page
   - Three clickable feat path cards that double as filters:
     - **DC 17 (gold)**: Max CHA 20, both slots on CHA, zero defense
     - **DC 16 (blue)**: Sweet spot CHA 18, one half-feat + one defense feat
     - **DC 15 (green)**: Full defense CHA 17, both slots on armor/concentration
   - Each card shows build count and can filter the grid when clicked
   - Build cards now have color-coded top borders matching their feat path
   - Each card shows "DC X path" label in the header
   - Added path filter to existing species/archetype/sort filter bar
   - Added "Clear path filter" button when a path is active

2. **Restarted Vite** after it crashed (had to use Start-Process with local vite.cmd since global npx resolved Vite 7 which needs Node 20.19+)

### Technical Details
- `getFeatPath()` function classifies builds by finalCha: ≥20 = dc17, ≥18 = dc16, else dc15
- FEAT_PATHS constant defines label, description, color, border color, and tradeoff text for each tier
- Moved declarations above BuildCard component since cards now reference them
