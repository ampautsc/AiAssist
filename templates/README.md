# Templates

This directory contains templates for common documents and tasks in the AI assistant system.

## Available Templates

### Task Templates
- **task-template.md**: Standard format for task documentation
- **complex-task-template.md**: Template for multi-session tasks

### Documentation Templates
- **skill-template.md**: Creating new skills
- **learning-entry-template.md**: Learning journal entries
- **decision-record-template.md**: Architecture/design decisions

### Workflow Templates
- **investigation-template.md**: Problem investigation workflow
- **code-review-template.md**: Code review checklist
- **feature-implementation-template.md**: New feature development

## Using Templates

1. Copy the appropriate template
2. Fill in all required sections
3. Customize for your specific needs
4. Save in the appropriate location
5. Update regularly

## Creating New Templates

When you notice a recurring pattern:

1. Document the standard structure
2. Include all necessary sections
3. Provide inline guidance
4. Add examples where helpful
5. Place in this directory
6. Update this README

## Template Guidelines

### Good Templates
- Clear section headings
- Inline instructions in [brackets]
- Examples of what to include
- Required vs optional sections marked
- Consistent formatting

### What to Template
- Recurring document types
- Standard workflows
- Common task structures
- Regular reporting formats
- Standardized processes

### What Not to Template
- One-off documents
- Highly variable content
- Creative work
- Exploratory investigations
- Context-specific items
