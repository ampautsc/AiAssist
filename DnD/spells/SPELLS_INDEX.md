## Spell Documentation Template

When adding new spells to the level files, use the following format to ensure we have all the details needed for character evaluation:

```markdown
### [Spell Name]
*Level [X] [School]*

- **Casting Time:** [e.g., 1 action, 1 bonus action, 1 reaction]
- **Range:** [e.g., 120 feet, Touch, Self]
- **Components:** [V, S, M (materials)]
- **Duration:** [e.g., Instantaneous, Concentration, up to 1 minute]
- **Classes:** [e.g., Wizard, Sorcerer, Warlock]
- **Tags:** [e.g., Damage, Control, Buff, Debuff, Utility, Healing]

**Description:**
[Full spell description text]

**At Higher Levels:**
[If applicable, what happens when cast with a higher level slot]

**Evaluation Notes:**
* [Pros/Cons for specific builds]
* [Synergies with class features or other spells]
```
