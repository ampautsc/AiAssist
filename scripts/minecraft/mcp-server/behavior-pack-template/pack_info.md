# MCP Control Pack

This behavior pack enables programmatic control of your Minecraft world through the MCP server.

## Features

- Real-time command execution from MCP server
- WebSocket communication bridge
- Helper functions for common operations
- Event handling and world monitoring
- In-game chat commands for testing

## Installation

1. Copy this pack to your Minecraft behavior_packs folder
2. Enable the pack in your world settings
3. Enable "Beta APIs" in experimental features
4. Start the MCP server
5. Join your world

## Usage

### In-Game Commands

- `!mcp status` - Check MCP connection status
- `!mcp <command>` - Queue a command manually

### From MCP Server

Send commands via REST API or Python client to control the world programmatically.

## Requirements

- Minecraft Bedrock Edition 1.20.0 or higher
- Beta APIs enabled
- MCP server running
